  • Client: Academic Project

  • Project URL:

  • Status: Upgrading UI with Responsiveness

An Online Doctor Appointment and Consultation System built using the MERN stack (MongoDB, Express.js, React, Node.js). The platform enables seamless doctor appointment scheduling, real-time video consultations, and digital prescriptions. Features include patient profile management, doctor availability tracking, and secure video calls for remote consultations. Integrated payment gateways for hassle-free billing and ensured data security with JWT authentication. Version control and collaboration were managed using GitHub. The system enhances accessibility, improves healthcare efficiency, and ensures a smooth user experience.

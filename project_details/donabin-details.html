Donabin Portfolio

Project information

  • Project duration: January, 2025 - Present
  • Project URL: GitHub Repository Link

Donabin Portfolio Web App is a platform developed for a US-based startup, focusing on managing donated clothes, shoes, and essential items. Built with a modern and responsive design, the application showcases the startup’s mission, impact, and donation process. Key features include donation tracking, inventory management, and seamless donor engagement. Developed with a user-friendly interface to enhance accessibility and transparency, ensuring a smooth experience for donors and administrators. The platform plays a crucial role in promoting sustainable giving and efficient resource distribution.

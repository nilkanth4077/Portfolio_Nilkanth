Project information

  • Project URL:

A full-stack E-commerce application using React for the frontend and Spring Boot for the backend. Integrated MySQL as the database and utilized Aiven services for cloud-based data management. Implemented a seamless shopping experience with cart management, checkout, and order processing. Managed version control and collaboration using GitHub. The platform ensures scalability, security, and a smooth user experience.
